# Business Skip Hire – React Redesign

This project is a modern, fully responsive redesign of the Business Skip Hire selection page using React and Tailwind CSS.

## ✅ Features

- Responsive grid layout for skip options
- Live API fetch with skip data
- Clickable skip cards with visual selection
- Clean, accessible UI/UX
- Tailwind CSS for styling
- Mobile & desktop ready

## 🧰 Stack

- React + Vite
- TailwindCSS
- Hosted via CodeSandbox
